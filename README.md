# explore-the-world
Explore the World is an interactive travel application built with JavaScript, HTML, CSS, and Vite. The application uses the REST Countries API to display information about countries around the world and allows users to save their dream destinations.

## Country Search

Search for a specific country using the search bar. The application performs a new API request when a search is made.

## Region Filter

Filter countries by region:

* Europe
* Asia
* Africa
* Americas
* Oceania

## Language Filter

Filter countries based on their official languages.

## Sorting

Sort countries by:

* A-Z
* Z-A
* Highest Population
* Lowest Population

## Country Details

Click "View More" to open a modal containing:

* Country flag
* Capital city
* Region
* Languages
* Currency
* Population
* Area
* Link to Google Maps

## Dream Destinations 🌴

Save countries to your personal Dream Destinations list.

Features include:

* Add countries
* Remove countries
* View saved countries
* Data persistence using Local Storage

### Responsive Design

The application is fully responsive and works on:

* Desktop
* Tablet
* Mobile devices
